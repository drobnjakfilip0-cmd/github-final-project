# Coursera-test
Simple Interest Calculator

This is a simple program that calculates simple interest based on the principal amount, interest rate, and time period.

Formula used:

Simple Interest (SI) = Principal (P) × Rate (R) × Time (T)

Features

Calculate simple interest quickly and easily.

Input principal amount, interest rate, and time in years.

Get the total interest amount.

How to Use

Run the program.

Enter the principal amount (the money you invest or borrow).

Enter the interest rate (as a decimal or percentage).

Enter the time period in years.

The program will calculate and display the simple interest.

Example
Principal: 1000
Rate: 5%
Time: 2 years
Simple Interest: 100
